# Water React App

A minimal React + Vite setup with a small zustand state example.

## Quick start

```bash
# From the project root
npm install react react-dom
npm install -D vite @vitejs/plugin-react

# Run the dev server
npm run dev

# Build for production
npm run build

# Preview the production build locally
npm run preview
```

Open the dev URL Vite prints (usually http://localhost:5173) in your browser.
